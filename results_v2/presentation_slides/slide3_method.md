# 슬라이드 3: 방법

## 제목
JuPedSim 기반 보행 시뮬레이션, 100회 시나리오

## 메시지
물리 기반 보행 모델과 게이트 선택 모델을 결합해 승강장에서 지상까지 전 구간을
재현합니다.

## 대상 공간
- 성수역 2층 대합실 **서쪽 절반** (50m × 25m)
- 게이트 7개, 계단 2개(승강장 연결), 에스컬레이터 2개(지상 출구 1번/4번)
- 성수역 안내도 실측 기반 좌표

## 모델 구성
| 계층 | 모델 | 근거 문헌 |
|---|---|---|
| 게이트 선택 (의사결정) | Gao 2019 LRP | Gao et al. 2019 |
| 보행 동역학 (전술) | CFSM V2 | Tordeux et al. 2016 |
| 게이트 통과 | 소프트웨어 FIFO 큐 + 서비스 시간 | Gao 2019, 실측 1.2~2.0초 |
| 에스컬레이터 | 소프트웨어 큐, 0.85초/명 | Cheung & Lam 2002 |

## 실험 설계
- 태그리스 이용률 p: **0.1 / 0.3 / 0.5 / 0.7 / 0.8** (5수준)
- 전용 게이트 수 k: **1 / 2 / 3 / 4** (4수준, 중앙 대칭 확장)
- 반복: 시드 **5회**
- **총 100 시나리오**, SIM_TIME 300초 = 열차 2편 처리

## 측정 지표
- 총 통행시간 (spawn → 지상 에스컬 도달)
- 구간별 시간: 게이트 대기 / 후처리(에스컬 포함)
- 통과율 = passed / spawned × 100%
- 구간별 밀도 (5초 간격 샘플)

## 한 문장 메시지
현실 역사의 보행 물리와 의사결정을 둘 다 반영한 시뮬레이션입니다.
