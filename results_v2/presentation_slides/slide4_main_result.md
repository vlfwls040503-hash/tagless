# 슬라이드 4: 결과 (핵심)

## 제목
이용률 × 배합 교호작용이 매우 강함 (η² = 0.62)

## 메시지
태그리스 이용률이 올라갈수록 최적 전용 게이트 수도 단조 증가합니다. 고정 배합은
비효율이며 가변 운영이 유효합니다.

## 최적 배합 (통과율 기준)

| 태그리스 이용률 p | 최적 전용 게이트 수 | 통과율 |
|---|---|---|
| 0.1 (10%) | **1개** | 100.0% |
| 0.3 (30%) | **2개** | 100.0% |
| 0.5 (50%) | **3개** | 100.0% |
| 0.7 (70%) | **4개** | 99.9% |
| 0.8 (80%) | **4개** | 99.6% |

**단조 증가 패턴 확인.** p ≈ k / N_gates 에 가까운 관계.

## 통계 근거

- 회귀 `avg_travel_time ~ p + config + p:config`: **R² = 0.696**
- 2-way ANOVA:
  - `config × p` 교호 효과: **η² = 0.62, F = 93.5, p < 0.001 (very large)**
  - `config` 주효과: η² = 0.16 (large)
  - `p` 주효과: η² = 0.01

## 역설 증거 (RQ2)

p와 맞지 않는 배합을 고르면 통과율 급락:
- p=0.1에서 **cfg 4** 선택 시 통과율 84.4% (cfg 1은 100%)
- p=0.8에서 **cfg 1** 선택 시 통과율 55.9% (cfg 4는 99.6%)
