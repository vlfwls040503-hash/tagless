# 연구 계획서

## 연구 제목
지하철 태그리스 개찰구 과도기 최적 게이트 운영 방안 연구

## 연구자
박필진 / 서울시립대학교 교통공학과 / 지도교수: 이동민

## 연구 목적
태그리스 개찰구 도입 과도기(태그/태그리스 혼재)에서 전용·겸용 게이트 배합에 따른 시스템 성능 변화를 분석하고, 시간대별 최적 운영 방안을 제시한다.

## 연구 배경
서울교통공사는 태그리스 개찰구를 단계적으로 도입 중이나, 과도기 동안 태그 이용자와 태그리스 이용자가 혼재하여 게이트 운영 효율이 저하될 수 있다. 특히 전용 게이트 분리가 항상 효율적인지, 혼입률에 따라 최적 배합이 달라지는지에 대한 정량적 근거가 부족하다.

## 연구 질문
1. 태그리스 이용자 혼입률 증가에 따른 병목 전이 현상 분석
2. 전용/겸용 게이트 분리의 효율성 역설 검증
3. 시간대별 가변 운영의 유효성 평가

## 연구 방법 (개요)
- **보행 시뮬레이션**: JuPedSim(CFSM V2) 기반 미시적 보행 모델 + 소프트웨어 큐 게이트 통과 모델
- **게이트 선택 모델**: Gao et al. (2019) LRP 모델 적용
- **시나리오 분석**: 태그리스 비율(20~80%) × 전용 게이트 수(0~3) × 시간대(첨두/비첨두) = 40개 시나리오

## 필요 데이터

### 1. 성수역 역사 시설 데이터 (설계도면)

본 연구는 승하차장(승강장)부터 대합실, 출구까지의 전체 보행 동선을 시뮬레이션으로 재현한다. 이를 위해 각 구간별 설계도면 및 치수 데이터가 필요하다.

| 구간 | 필요 데이터 | 시뮬레이션 활용 |
|------|-------------|-----------------|
| **승강장** | 승강장 길이, 폭, 계단 진입구 위치 | 하차 승객의 초기 위치 분포 및 계단까지의 보행 거리 산정 |
| **계단 (승강장→대합실)** | 계단 폭, 높이(층고), 위치, 개수 | 계단 방출율 산정 (Weidmann 1993), 보행자 도착 시간 모델링 |
| **대합실** | 평면도 전체 치수, 벽체·기둥·구조물 배치 | 보행 가능 영역(walkable area) 구축, 장애물 회피 경로 반영 |
| **개찰구(게이트)** | 게이트 수, 통로 폭, 하우징 폭, 게이트 간 간격, 배리어 길이 | 게이트 통과 모델 파라미터, 대기열 공간 산정 |
| **출구 (대합실→지상)** | 출구 위치, 폭, 계단/에스컬레이터 유무 | 게이트 통과 후 퇴장 경로 및 후속 병목 분석 |

> 설계도면이 최선이며, 도면 확보가 어려울 경우 현장 실측으로 대체할 수 있음

### 2. 수요 데이터
| 항목 | 상세 내용 | 용도 |
|------|-----------|------|
| 역별 시간대별 승하차 인원 | 성수역 08-09시(첨두), 14-15시(비첨두) 등 | 도착 수요 산정 (열차당 하차인원) |
| 열차 운행 간격 | 시간대별 배차 간격 | 도착 모델 열차 간격 파라미터 |

> 수요 데이터는 공공데이터포털(서울교통공사 역별 시간대별 승하차인원)에서 확보 가능

### 3. 우이신설선 현장 영상 촬영

우이신설선은 현재 태그리스 게이트를 실제 운영 중인 노선으로, 본 연구의 시뮬레이션 모델을 검증하기 위한 실측 데이터를 수집하고자 한다.

**촬영 목적**
| 실측 항목 | 내용 | 시뮬레이션 검증 대상 |
|-----------|------|----------------------|
| 태그리스 게이트 통과 시간 | 이용자의 게이트 진입~통과 소요 시간 | 서비스 시간 모델 (현재 1.2s 가정) 검증·보정 |
| 태그 게이트 통과 시간 | 교통카드 태그 이용자의 통과 소요 시간 | Gao et al. (2019) 실측값(평균 2.0s)과 비교 |
| 게이트 선택 행태 | 보행자의 게이트 접근 경로, 재선택(jockeying) 빈도 | LRP 게이트 선택 모델 파라미터 검증 |
| 대기열 형성 패턴 | 게이트별 대기열 길이 변화 | 큐 모델 검증 |

**촬영 계획**
- **대상 역사**: 우이신설선 태그리스 게이트 운영 역 1~2개소
- **촬영 시간대**: 첨두(08-09시) 및 비첨두(14-15시), 평일 1~2일, 각 2~3시간
- **개인정보 보호**: 얼굴 비식별 처리 예정, 연구 목적 외 사용 금지
- **촬영 위치**: 대합실 상부 고정 촬영 (보행 동선 전체 조감)
