# 태그리스 전용 게이트 분리 배치의 통행비용 절감 효과 분석 — 성수역 보행 시뮬레이션

## 1. 연구 배경

- 서울교통공사, 1~8호선 태그리스 시스템 도입 추진 중 (시범사업 진행)
- 현재 태그리스 게이트는 **태그/태그리스 겸용**으로 운영 (역당 1~2개)
- 나머지 게이트는 기존 태그(NFC) 전용
- 태그리스 이용자 증가가 예상되며, 통행비용 절감 효과는 이미 검증됨

## 2. 문제 제기

- 현행 겸용 게이트에는 태그 이용자(멈춰서 탭)와 태그리스 이용자(연속 통과)가 **혼재**
- 태그리스 이용자 비율이 높아질수록:
  - 겸용 게이트에서 태그 이용자로 인해 태그리스 이용자의 **대기행렬 발생**
  - 태그리스의 연속 통과 이점이 상실 → **통행비용 증가**
- 기존 겸용 게이트를 **태그리스 전용**으로 전환하면, 태그리스 이용자가 끊김 없이 통과 가능
- 그러나 전용 게이트 전환의 **정량적 효과**에 대한 근거가 부재

## 3. 연구 목적

현행 겸용 게이트 운영(기본 시나리오)과 태그리스 전용 게이트 분리 배치(개선 시나리오)의 통행비용을 비교하여, 전용 게이트 전환의 효과를 정량적으로 검증한다.

## 4. 연구 방법

### 4-1. 시뮬레이션 모델

- **도구**: PTV Vissim / Viswalk (학생 버전)
- **모델**: Viswalk 내장 Social Force Model (SFM) 기반 보행 시뮬레이션
- **대상역**: 성수역 (2호선)
  - 환승 없음, 역사 구조 단순
  - 승객/게이트 비율 상위 (일평균 105,574명 / 39개 게이트 = 2,707명/게이트)
- **공간 범위**: 게이트 → 대합실 → 계단/에스컬레이터

### 4-2. 성수역 게이트 현황

| 구분 | 수량 |
|------|------|
| 턴스타일 개집표기 | 31대 |
| 우대자용 개집표기 | 8대 |
| **합계** | **39대** |

- 현재 태그리스 겸용: 1~2대 (나머지는 태그 전용)

### 4-3. 이용자 유형 및 게이트 서비스 시간

| 이용자 유형 | 통과 방식 | 서비스 시간 |
|------------|----------|-----------|
| 태그 이용자 | 정지 → 카드 탭 → 문 열림 → 통과 | 약 2~3초 |
| 태그리스 이용자 | 감속 없이 연속 통과 | 약 1초 이하 |

### 4-4. 시나리오 설계

#### 기본 시나리오 (현행)
- 태그 전용 게이트 + **겸용 게이트 1~2대**
- 겸용 게이트에 태그/태그리스 이용자 혼재

#### 개선 시나리오
- 기존 겸용 게이트 1~2대를 **태그리스 전용**으로 전환
- 나머지 게이트는 태그 전용 (현행 유지)
- 추가 인프라 변경 없이 **게이트 설정만 변경**하는 현실적 방안

#### 공통 변수: 태그리스 이용자 비율(p)
- 전체 이용자 중 태그리스 비율: 10%, 20%, 40%, 60%, 80%
- 비율별로 기본/개선 시나리오 각각 시뮬레이션

### 4-5. 게이트 선택 의사결정 모델

보행자의 게이트 선택은 **이산선택모형(Discrete Choice Model)**으로 모사:

- Viswalk의 **Partial Route + Route Choice** 기능 활용
- 보행자가 게이트별 **이동거리 + 대기열 길이 + 서비스 시간**을 고려하여 확률적으로 선택
- 효용함수: `U_gate = β₁ × 거리 + β₂ × 대기열 길이 + β₃ × 서비스 시간`

이를 통해 겸용 게이트의 태그리스 이용자 비율을 **외생적으로 가정하지 않고, 시뮬레이션 결과로 산출**

### 4-6. 입력 데이터

| 데이터 | 출처 | 비고 |
|--------|------|------|
| 성수역 시간대별 승하차인원 | 서울교통공사 공공데이터 | 첨두/비첨두 구분 |
| 게이트 수 및 유형 | 서울교통공사 시설물 현황 | 턴스타일 31 + 우대자 8 |
| 태그 통과시간 | 현장 실측 또는 문헌 | 약 2~3초/인 |
| 태그리스 통과시간 | 문헌값 | 약 1초 이하/인 |
| 역사 평면도 | 서울교통공사 도면 또는 현장 측정 | Viswalk 공간 구현용 |
| 계단/에스컬레이터 용량 | TCQSM 문헌값 | |

### 4-7. 평가 지표

- 게이트 앞 **평균 대기시간**
- 게이트 앞 **최대 대기행렬 길이**
- **총 통과시간** (게이트 진입 → 계단 도달)
- 게이트-계단 구간 **보행자 밀도** (인/m²) 및 LOS

## 5. 기대 결과

1. **기본 vs 개선 시나리오 통행비용 비교**
   - 태그리스 비율별 대기시간, 통과시간 차이 정량화
   - 태그리스 비율이 높아질수록 전용 게이트 전환 효과 증가 예상

2. **전용 게이트 전환의 임계점 도출**
   - "태그리스 비율 X% 이상에서 전용 게이트 전환이 유의미한 효과"

3. **겸용 게이트 내 혼합 비효율 정량화**
   - 겸용 게이트에서 태그 이용자로 인한 태그리스 이용자 대기시간 산출

4. **정책 제언**
   - 현 인프라에서 게이트 설정 변경만으로 달성 가능한 통행비용 절감 효과 제시

|------|------|

| 강남역 ABM (MDPI 2023) | Agent-based model로 강남역 보행 분석 |

### 연구 차별성
- 태그리스 **전용/겸용 분리 배치**의 통행비용 비교 → 선행연구 없음
- 게이트 **서비스율 차이**가 혼합류에 미치는 영향 → 연구 없음
- **현실적 제약**(기존 인프라 활용, 1~2대 전환) 하의 최적화 → 연구 없음
