# 회의록 2026-03-17 (3주차 발표 후)

## 교수님 컨펌 완료

주제: 지하철 역사 내 태그리스 개찰구 운영 방안

## 변경/추가 사항

### 1. 현장조사 역 변경: 성수역 → 우이신설선
- 성수역에는 태그리스 게이트가 아직 없음
- 우이신설선은 2023년 세계 최초 태그리스 상용화 노선 → 실측 데이터 확보 가능
- 우이신설선에서 파라미터 실측 → 성수역 시뮬레이션에 적용 (transferability)

### 2. 현장조사 측정 항목
| 항목 | 피크 | 비피크 | 비고 |
|------|------|--------|------|
| 태그 통과시간 | O | O | 카드 대는 순간~게이트 통과 |
| 태그리스 통과시간 | O | O | 게이트 진입~퇴출 |
| 접근 보행속도 | O | O | 게이트 전 1m 구간 |
| 퇴출 보행속도 | O | O | 게이트 후 1m 구간 |
| 대기행렬 길이 | O | O | 게이트 앞 대기 인원 |
| 태그리스 혼입률 | O | O | 태그리스/전체 비율 |
| 게이트별 도착률 | O | O | 5분 단위 통과 인원 |

### 3. 측정 방법
- 스마트폰 60fps 촬영 (개찰구 측면)
- 바닥 타일 or 테이프로 기준점(전 1m, 후 1m) 설정
- 분석: Kinovea (반자동) 또는 YOLO+DeepSORT (자동화)

### 4. 시나리오 확장
- 기존: 겸용 vs 분리 (혼입률별)
- 추가: **태그리스 혼입률에 따른 전용 게이트 최적 개수** 도출
- 추가: **피크 vs 비피크 시간대별** 비교

### 5. 시간대
- 피크: 출근 7:30~9:00 / 퇴근 18:00~19:30
- 비피크: 13:00~14:30
- 각 시간대 최소 30분 연속 촬영

## 핵심 아날로지
고속도로 하이패스 전용 차로 분리 → 지하철 태그리스 전용 게이트 분리 (과도기 전략)
최종적으로는 게이트 프리(게이트 없이 통과)가 목표이나, 기술적 한계(실내 측위 등)로 과도기가 길 것
